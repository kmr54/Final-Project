Projects I completed this semester
Project Links What I Learned
Chapter 1 This was the first website I created. In order to build the Curbside Thai menu, I learned how to work with lists, id and class attributes, tags, and links.
Chapter 2 The Tri and Succeed Sports webpage helped introduce me to CSS. I learned how to use contextual selectors in order to access tags, ids, and classes to add styles to, as well as how to change the initial look of HTML elements such as lists and images. I also learned about the difference between inline styles and external style sheets.
Chapter 3 While working on the Pandaisia Chocolates webpage, I learned about webpage layouts and the grid system. In order to make webpages more visually appealing and organized, the space and positioning of the elements needs to be set. This topic is definitely something I still need to work on understanding, but I do recognize it is a very important one to grasp in order to be a successful web developer.
Chapter 4 With the Tree and Book project, I learned about figures and figcaption, as well as how to add graphics to images. I adjusted an image's rotation, color, and opacity. I also learned about image maps, which I thought were very cool.
Chapter 5 I relied heavily on the Trusted Friends Daycare tutorial for this project. It helped me with creating media queries and having a responsive navbar. This tutorial helped me, but responsive web design is still something I have yet to fully understand even after watching and reading many tutorials.
Chapter 6 The DLR website taught me how to create and style a table. I learned how to break the table down into row and column groups to help apply certain styles and features to parts of the table, in addition to how to span content over a number of rows or columns.
Chapter 7 I learned about forms while working on the Red Ball Pizza website. I built a form that had two fieldsets and used a variety of inputs. I also learned a little JavaScript and client-side validation.
Chapter 8 The Cinema Penguin website was my favorite to build. I used the navbar style effect from this website for the desktop display of my navbar. Additionally, I utilized what i learned about video and audio tags for that section of my website.
Chapter 9 The New Year's Clock website taught me how to incorporate JavaScript into a website. I created a countdown clock using Date objects.
Chapter 10 The final website I worked on was the Lyman Hall Theater calendar. It taught me more about JavaScript and how to build a table using it. I was introduced more to building functions in JavaScript and how to use loops.
